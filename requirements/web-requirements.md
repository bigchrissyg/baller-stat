# Baller Stats - Histon Hornets Blue U12
This should be a dashboard website for managing the team's season detailing the players, the fixture (matches) and the playing statistics.

As well as being a dashboard for viewing data, it should also all data entry and correction.

## Homescreen
The home screen should show the highlevel statistics for the season such as:
- Matches played
- Total minutes played
- Total goals scored
- Total goals conceded
- Total quarters with a clean sheet
- Any other statistics which the indusry would typically capture

It should also have a digest of all the season fixures in reverse chronological order

Each fixture should be clickable and go into a detailed match view.

This should be filterable by fixture type (friendly, league etc)

There should be an option to add a new fixture including detailing all the properties. Upon filing that in, you are taking to the detailed match view where you can assign players and their intended positioning.

## Match view
This is the detail of a specific match. Here you should see the list of players and the ones assigned to the match. Next to them you should see a matrix of the positions they played for each 1/8th of a match. 

For matches that have been played, there should be all the match events details (times of goals, who scored them and who assisted)

Who are the star players and why.

There should be match specific statistics which show:
- Total goals scored
- Total goals conceded
- Total quarters with a clean sheet
- Any other statistics which the indusry would typically capture

This should have an edit mode which allows you to change the playing time and positions and match events.

Clicking on a player should go to the individual player profile screen.

## Player profile
Player profile should display their name and all the player stats including all the positions they played and their time spent in those postions.
